Detailed Description

A layout container that arranges child widgets along a given axis.

Participates in two-pass layout via IContainer. Supports optional scrolling along the main axis.

Member Function Documentation

◆ Render()

void ConsoleForge.Widgets.Container.Render ( IRenderContext ctx)

Renders each child into a sub-region computed at render time from IRenderContext.Region.

This avoids dependency on ResolvedLayout child regions, which are only valid when the container sits directly at the layout root (not inside a BorderBox or other non-IContainer wrapper).

Implements ConsoleForge.Layout.IWidget.
